# Chat Application with WebSocket Support

This is a simple chat application built using Java and Spring Boot, which enables real-time communication between users in a chat room. Users can join the chat room, send messages, and receive notifications when someone enters or leaves the room. The application leverages WebSocket technology to achieve real-time functionality.

## Features

- Real-time chat with WebSocket support.
- Notifications for user entry and exit.
- Supports any number of concurrent users.
- Simple and user-friendly interface.

## Technologies Used

- Java
- Spring Boot
- WebSocket (via Spring WebSocket)
- Thymeleaf (for the web UI)
- Maven (for project management)
- Bootstrap (for styling)

## Prerequisites

Before you begin, ensure you have met the following requirements:

- Java Development Kit (JDK) installed
- IDE of your choice (e.g., IntelliJ IDEA, Eclipse) for development
- Basic knowledge of Java and Spring Boot

## Getting Started

1. Clone the repository to your local machine:
2. Open the project in your IDE.
3. Configure the application properties in src/main/resources/application.properties as needed, especially the database and server settings.
4. Run the application
5. Access the application in your web browser at http://localhost:8080.
